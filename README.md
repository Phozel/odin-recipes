# odin-recipes

 ## What is this project?
This is the first project given while working through the The Odin Project curriculum. 

The goal is to create a website consisting of a main index page that links to a few recipes.

## What skills will this project demonstrate?
The aim of the project is to practice basic html. As such, when it is finished it will show that I have a basic understanding of html and how it functions.

## Post Project Reflections
This project was fun, even though I had to scrap the lasagna page as I do not know the recipe for that. In the middle of working on it I also decided to install ZorinOS on my laptop and get used to using Linux, and not through a Virtual Machine.

By sitting down and writing the HTML in this way, without using any external help to create a very bare-bones website I do not think I learned more than I had when going through the sections in TOP. However, what I did gain was more experience writing the language and thus a more instinctive understanding of it. But I definitely have more to learn given how early on in my journey this is and as such I cannot wait to move on to CSS and learn how to make my sites look great.

Furthermore, the process of exploring Linux and different distributions available was fun to do as well. It has given me a deeper appreciation for Linux and its multiple different flavours. 

If you reading this, is a student of TOP, and is using a VM, I cannot recommend at least exploring what Linux distributions are available to see if any of them are for you. Who knows, maybe you will love it?
